## 🪄 Features

- 🚀 一端发布内容，另一端无需登录直接复制
- 📱 支持手机扫码复制
- ⏱ 可选有效期1-5小时
- 🗑 指定时间内自动删除内容
- ❌ 可删除自己发布的内容
- 📈 内容复制统计
